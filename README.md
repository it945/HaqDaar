# HaqDaar (حقدار)

Empowering transparent aid distribution through secure identity and instant verification.

This project was migrated from a Flutter repository to a high-performance **React + Vite** web application, optimized for use in AI Studio.

## Features

- **🛡️ Secure Identity**: Digital ID card with verifiable QR codes.
- **📷 Scan & Pay**: Instant aid redemption at authorized merchants.
- **📊 Usage Insights**: Visual analytics of aid consumption using Recharts.
- **✨ HaqDaar AI**: Integrated Gemini-powered assistant for beneficiary support.
- **📱 Fluid UX**: Mobile-first design with smooth Framer Motion animations.

## Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Charts**: Recharts
- **AI**: Google Gemini API (@google/genai)
- **Icons**: Lucide React

## Development

Run the development server:

```bash
npm run dev
```

The application runs on port 3000.
